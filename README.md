## Запуск проекта

Для запуска проекта:
1. docker compose build
2. docker compose up

После запуска, вы сможете делать запросы по адресу:
http://localhost:3002/getOrder?uid=


UID заказа можно получить по запросу:
http://localhost:3002/getTestOrder
